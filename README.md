# Игра на реакцию для двух игроков на ATmega328P

DIY-гаджет: игра на реакцию для двух игроков. После случайной задержки загорается центральный зелёный светодиод — кто быстрее нажмёт свою кнопку, тот выигрывает раунд. Первый до 3 очков выигрывает матч.

Проект на "голом" ATmega328P (без Arduino-платы). Код написан в Arduino IDE (только core-функции, без библиотек), прошивка HEX-файлом через программатор (T48).

## Версии
- **v1.0** : простая индикация — LED1 (P1) и  LED2 (P2) мигают при победе в раунде. При победе в матче — быстрое мигание индикатора победителя.
- ![Плата сверху](images/v1/IMG_4161.heic)
- **v2.0** (в разработке): улучшенная индикация — по три LED на игрока . Индикаторы счёта зажигаются последовательно и остаются гореть. При победе в матче — красивая поочерёдная анимация всех трёх индикаторов победителя.

## Механика игры (общая для обеих версий)
- Стартовый сигнал: 3 коротких моргания центрального  LED.
- Случайная задержка 1–5 секунд.
- Анти-чит: более 2 нажатий кнопки во время ожидания — игрок блокируется на раунд.
- После загорания центрального LED — 5 секунд на реакцию.
- Матч до 3 очков.
- Автоматический сброс после окончания матча.
